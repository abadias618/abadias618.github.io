Decision Trees - ID3 Algorithm

The ID3 algorithm was implemented first with some pruning, it restricted the levels of the tree by not alowing repeats in each path. However, for these 3 datasets, allowing repeats and therefore longer paths to each leaf, higher results were achieved. Recursion (Pre-Order Traversal) was used to visit all leaves after getting the lowest entropy attribute, to determine the root.

Neural Network Binary Classifier

The Wisonsin Breast Cancer Data was used to train a Neural Network which classifies if tumors are benign or malicious.

Code
Neural Network Binary Classifier

Min-Max scaling was needed because of the sparsity of the Wisconsin Breast Cancer Dataset. Tensorflow's Keras were used to train the model.



Three "Dense" layers of "ReLu" activation functions were used, and a final level with the "Sigmoid" activation function. With an "rmsprop" optimizer and a "binary cross entropy" loss function, in 500 epochs 0.96% accuracy was achieved.
